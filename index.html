Sistema de Gestión Médica y Análisis de Datos (SGMAD)

Empresa:

Hospital Santa Bárbara Integrado.

Ciudad:

Santa Bárbara

Descripción:

Desarrollo de un sistema integral para la gestión de pacientes y análisis de datos clínicos. El sistema incluye módulos para la administración de historiales médicos, gestión de empleados, generación de informes estadísticos en pdf o archivos Excel y la recuperación de contraseñas por medio de WhatsApp o correo electrónico

El sistema está diseñado para mejorar la eficiencia operativa de procesos en un área específica dentro del hospital, facilitando la gestión de pacientes y la toma de decisiones basadas en datos correctos. El análisis de datos se hace con herramientas de visualización de datos como tablas y gráficos dinámicos en Excel.

Tecnologías:

  1. Java
  2. JavaFX
  3. MySQL
  4. JasperReports

Estado:

En producción y en uso activo en el hospital.

Sistema Web para Gestión de Reservas y Servicios Turísticos

Empresa:

Proyecto Personal – Complejo de Cabañas "La Aguja"

Ciudad:

San Luis, Santa Bárbara

Descripción:

El sitio web ofrece un sistema de reservas y gestión de servicios turísticos para el Complejo de Cabañas "La Aguja". Los usuarios pueden explorar las cabañas disponibles, ver detalles y fotos, y realizar reservas y reseñas en línea de manera sencilla y segura.

El sistema permite a los editores la gestión de correos y responder a las reseñas de los clientes registrados en el sitio web y la gestión de la disponibilidad de servicios adicionales ofrecidos por el complejo, a través de diferentes paneles de administración.

Tecnologías:

  1. JavaScript
  2. jQuery
  3. HTML5
  4. CSS
  5. Bootstrap
  6. MongoDB
  7. Node.js
  8. Express.js

Estado:

En análisis y desarrollo.
